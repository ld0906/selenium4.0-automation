# 安装Python 虚拟环境

## 1: 安装 virtualenv 软件包
` pip install virtualenv`  or ` pip3 install virtualenv`

## 2: 安装虚拟环境
`virtualenv <virtual environment name >`

## 3: 安装自动化测试所需要的类库
`pip install -r requirements.txt`
